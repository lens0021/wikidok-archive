## 위키독 보존

이 리포지토리는 위키독의 문서를 일부 보존하고 있으며 보존된 데이터를 활용할 수 있는 방법을 마련 중입니다.

### 보존 위키와 각 저작권

- 아름드리위키 (CC BY-SA 3.0)
- 비건편의점 WiKi
- 여성위키

### 주의사항

- 정적 웹사이트를 만드는 것은 목표로 하고 있지 않습니다(트래픽 문제).
